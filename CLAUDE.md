# Insurance Optimizer

## Overview
Insurance policy optimization tool. Monorepo with a React frontend and Express backend that analyzes policies and recommends optimal coverage.

## Classification
- **Tier:** 2 — Interactive
- **Company:** Freebridge

## Tech Stack
- **Client:** React 19, React Scripts 5, Tailwind CSS 3
- **Server:** Express 4, Node.js
- **Package Manager:** npm

## Key Commands
- `npm start` (client) — Start React dev server
- `npm start` (server) — Start Express server
- `npm run build` (client) — Production build

## Project Structure
- `client/` — React frontend (Create React App)
- `server/` — Express backend API
